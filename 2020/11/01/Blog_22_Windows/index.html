Blog_22_Windows

知识点：windows hosts 文件

    1. C:\Windows\System32\drivers\etc\hosts

知识点：windows scp

    1. 介绍
        # 都只能在 windows 上执行命令
        # 推荐使用 cmd 而不是 powershell，因为可以使用 windows 变量
    1. windows to linux
        # scp %HOMEPATH%\Desktop\1.txt root@<hostname>:~
    2. linux to windows
        # scp root@~/1.txt %HOMEPATH%\Desktop
            - 复制到桌面

知识点：windows 常用变量

    1. 介绍
        # 网址
            - http://www.wilsonmar.com/1envvars.htm    
        # 使用和查看
            - 只能在 cmd 中使用，不能在 powershell 中使用
            - 使用 echo 查看变量的值

知识点：windows 查看文件的 MD5 SHA1 SHA256 校验值

    1. 命令格式
        # certutil -hashfile file_name MD5  
        # certutil -hashfile file_name SHA1  
        # certutil -hashfile file_name SHA256
