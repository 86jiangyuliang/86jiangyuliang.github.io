知识点：notepad 快捷键

    1. Alt + Shift + Arrows|鼠标    <== 纵向选择
    2. 标签页
        # Ctrl+N                    <== 创建新标签页
        # Ctrl+PageUp/Dowd            <== 选择标签页
        # Ctrl+W                    <== 关闭标签页
    3. Ctrl + 鼠标滚轮                <== 改变字体大小
    4. Shift + 鼠标滚轮                <== 水平滑动

知识点：显示格式字符

    1.视图 => 显示符号
        # 显示空格与制表符
        # 显示行尾符
        # 显示所有字符

        # 显示换行符
